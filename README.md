# Sentiment-Analysis
Sentiment analysis of customer reviews using Python, Pandas, TextBlob, and Matplotlib with data visualization of sentiment distribution.
# Sentiment Analysis Using Python

## Project Overview
This project performs sentiment analysis on customer reviews using Python. It analyzes the text of reviews and classifies them into Positive, Negative, or Neutral sentiments using the TextBlob library.
## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib
- Google Colab
## Dataset
The project uses a CSV file containing customer reviews. The review text is stored in a column named "Review".
## Working Process
1. Upload the reviews dataset.
2. Load the CSV file using Pandas.
3. Analyze each review using TextBlob.
4. Classify sentiments as Positive, Negative, or Neutral.
5. Store the results in a new CSV file.
6. Visualize the sentiment distribution using a bar chart.
## Output
The project generates:
- sentiment_results.csv containing the original reviews and their predicted sentiments.
- A bar chart showing the number of Positive, Negative, and Neutral reviews.
## How to Run
1. Open the notebook in Google Colab.
2. Upload the Reviews.csv file.
3. Run all cells.
4. View the sentiment results and generated graph.
